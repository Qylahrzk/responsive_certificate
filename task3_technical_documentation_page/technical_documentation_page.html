Introduction

HTML stands for HyperText Markup Language.

It is the standard markup language for creating Web pages.

HTML describes the structure of a Web page.

It consists of a series of elements.

HTML elements tell the browser how to display the content.

<!DOCTYPE html> <html>...</html>
  • Basic Structure
  • Head and Body
  • Metadata
Elements

HTML elements are the building blocks of HTML pages.

They are represented by tags such as <p> or <div>.

<p>This is a paragraph.</p> <div>A division block.</div>
  • Paragraphs
  • Headings
  • Divisions
Attributes

Attributes provide additional information about HTML elements.

They are always specified in the start tag.

<a href="https://example.com">Visit Example</a>
  • href for links
  • src for images
  • alt for image descriptions
Forms

Forms are used to collect user input.

They contain input elements like text fields and submit buttons.

<form>...</form> <input type="text">
  • Text inputs
  • Radio buttons
  • Checkboxes
Media

HTML allows you to embed images, audio, and video on webpages.

Use the <img>, <audio>, and <video> elements for media.

<img src="image.jpg" alt="description">
  • Images
  • Audio files
  • Video files
